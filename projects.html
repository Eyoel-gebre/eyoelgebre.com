Movie review binary classifier
August 2022

• Trained a Dense neural network to identify whether a movie review is positive or negative based on its text.
• Utilized the Keras deep learning API along with TensorFlow to train the model on a GPU using google Collaboratory
• The model was able to achieve ~87% accuracy on test data and was trained on data from the Keras IMDB data set.
